# 职工管理系统 java81

## 项目概述

职工管理系统是一套适用于企业、单位进行人事管理的系统。该系统基于前后端分离的模式开发，使用Java、SpringBoot和Vue等技术栈。系统提供基础的人员管理和考勤功能，并可根据客户需求定制增加协同过滤推荐算法、支付宝沙箱环境、AI模型对话等高级功能。

## 技术栈

- 前端：Vue
- 后端：Java、SpringBoot
- 数据库：Mysql

## 系统角色与功能模块

### 用户功能模块

- **首页**：查看个人信息，退出账号
- **公告列表**：查看公告编号、公告标题、公告发布时间等
  - 公告编号
  - 公告标题
  - 公告发布时间
- **奖惩信息**：查看序号、编号、类型、日期、原因、金额
  - 序号
  - 编号
  - 类型
  - 日期
  - 原因
  - 金额
- **考勤信息**：查看类型、状态、日期、序号、编号
  - 类型
  - 状态
  - 日期
  - 序号
  - 编号
- **个人信息**：查看和修改个人资料
- **修改密码**：修改个人密码

### 超级管理员功能

- **首页**：查看个人信息，退出账号
- **部门管理**
  - 部门列表
  - 添加部门
- **职工管理**
  - 职工列表
  - 查看所有职工、离职职工
- **考勤管理**
  - 正在审核列表
  - 审核通过列表
  - 未通过审核列表
  - 添加考勤
- **添加职工**
  - 职工编号
  - 职工名称
  - 性别
  - 身份证号
  - 地址
  - 部门
  - 职工照片
- **奖惩管理**
  - 奖惩列表
  - 添加奖惩
- **公告列表**：查看公告内容
- **合同管理**
  - 合同列表
  - 添加合同
- **修改密码**：修改管理员密码
